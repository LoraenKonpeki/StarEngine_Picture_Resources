## 字段说明

### Full（立绘）

立绘文件存储于 `Full/{编号}/` 目录下，文件命名规则如下：

| 文件类型 | 命名格式 | 说明 |
|----------|----------|------|
| 默认立绘 | `UT_Hero_Card_{编号}.png` | 角色默认立绘 |
| 皮肤立绘 | `UT_Hero_Card_{编号}_{序号}.png` | 皮肤/时装立绘（序号 01~05） |
| 羁绊立绘 | `UT_Hero_Card_{编号}_Max.png` | 好感度满级时的羁绊立绘（含变体） |
| 安全版 | `UT_Hero_Card_{编号}*_sfw.png` | 无背景/透明化版本 |

### Emoji（表情）

表情文件存储于 `Emoji/{编号}/` 目录下，每个角色的表情图片以编号命名。

### Pixel（像素头像）

像素头像文件存储于 `Pixel/{编号}/` 目录下，文件命名格式为 `UT_AccountHeadShot_709{后缀}.png`。

三位数角色（101~127）对应的后缀为 `01`~`27`（即角色编号减 100）；联动角色（301~304）直接使用完整编号作为后缀。

### Sprite（对局贴图）

对局内人物贴图存储于 `Sprite/{编号}/` 目录下，文件命名格式为 `UT Hero LevelUp {编号}.png`。
